## Register User A.P.I
- <b>/v1/register</b>
### Register Success
```
Response Code - 201
``` 

`Request Sample`
```json
{
  "name": "Test User",
  "email": "test.user@gmail.com",
  "password": "test_1234"
}
```

`Response Sample`
```json
{
    "message": "Success created user",
    "data": {
        "ID": 3,
        "name": "Test User",
        "email": "test.user@gmail.com"
    }
}
```
![Alt register-success](/img/register_success.png)
